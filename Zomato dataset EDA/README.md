# Zomato Bangalore Analysis

Exploratory Data Analysis on a Zomato Bangalore restaurant dataset using Python.
| Dataset url: https://www.kaggle.com/datasets/ruchikakumbhar/zomato-dataset

## Dataset
- **Source:** Zomato Bangalore dataset
- **Records:** 148 restaurants, 7 features
- **Features:** Restaurant name, online order, book table, rate, votes, approx cost, listed type

## Questions Explored

| # | Question |
|---|---|
| Q1 | What is the distribution of restaurant ratings? |
| Q2 | Do restaurants with online ordering have higher ratings? |
| Q3 | What proportion of restaurants offer online ordering? |
| Q4 | How do ratings vary across different restaurant types? |
| Q5 | Which restaurant types are most common? |
| Q6 | What are the most popular cuisines? |
| Q7 | What percentage of restaurants per type have ratings above 4.0? |
| Q8 | How does average cost vary across restaurant types? |

## Key Findings
- Restaurants **with online ordering** have higher average ratings (3.86) vs without (3.49)
- Only **39% of restaurants** offer online ordering — offline dining still dominates
- **Casual Dining** is the most common restaurant type
- **North Indian** cuisine is the most popular
- Revenue is right-skewed — most restaurants are budget-friendly while a few are premium
