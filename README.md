# EDA Projects

A collection of Exploratory Data Analysis projects on real-world datasets using Python.
Each project explores a different domain through data cleaning, visualization, and insight generation.

## Projects

| Project | Dataset | Records | Notebook |
|---|---|---|---|
| Superstore Sales Analysis | Retail Store Chain | 118 stores | [View](./Superstore%20dataset%20EDA/Sales_store.ipynb) |
| Zomato Bangalore Analysis | Zomato Bangalore | 148 restaurants | [View](./Zomato%20dataset%20EDA/Zom_eda.ipynb) |

## Skills Demonstrated
- Data cleaning and null value handling
- Exploratory analysis through business questions
- Data visualization using Matplotlib and Seaborn
- Drawing conclusions from real-world datasets

## Tools Used
- Python, Pandas, NumPy
- Matplotlib, Seaborn
- Jupyter Notebook
